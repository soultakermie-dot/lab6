# Лаб 6

Консольний додаток перекладача з англійської мови на українську.

## Опис

Програма реалізує перекладач з використанням HashMap для зберігання словника англійських та українських слів.

## Структура проекту

- `Translator.java` - клас перекладача з методами для додавання слів та перекладу фраз
- `TranslatorApp.java` - консольний додаток з інтерфейсом користувача

## Використання

Запустіть `TranslatorApp.java` для роботи з перекладачем.
